¿Qué es DynHost?

DynHost es un sitio que permite la creación de nombres de máquina, de modo que, si tienes un servidor o una máquina en Internet con IP dinámica (que cambia cada poco tiempo), puedes crear un nombre que se vaya actualizando con los cambios de IP que vaya teniendo tu máquina de cara a Internet.

Esto es útil para:

  • Tener localizada siempre la IP de tu casa.
  • Un nombre para acceder a tu servidor casero.
  • Un subdominio temporal para montar y presentar una web.

Uso profesional

DynHost ha agregado funcionalidades específicas para permitirte gestionar dominios reales, no solo dynhost.es. En general:

Dominios: Tu nombre en Intenet

¿Necesitas comprar un dominio y gestionar los registros a todos los niveles y sin restricciones? ¡Aquí puedes! Entra con tu cuenta y elige el dominio que quieras.

Buzones de Correo Electrónico

Tu dominio con buzones de correo electrónico incluidos. Podrás configurar buzones de correo accesibles mediante protocolo POP3 o IMAP o empleando nuestro sistema de webmail.

Alojamiento para tu web

Adquiriendo un dominio tendrás acceso a un espacio para tu web. Podrás instalar cualquier sistema como Wordpress, Drupal, Prestashop, SMF, ... de forma muy sencilla con un instalador automático, subiendo ficheros vía FTP o a través de git.

Crear Redirecciones Web

Si tienes tu página creada en una herramienta de empresa externa y quieres que tu nuevo dominio la emplee, simplemente configurando esta redirección, el sitio web recibirá las peticiones desde el dominio que hayas adquirido. Incluso si son varios y quieres unificarlos bajo varias rutas.

Manejar Redirecciones de Correo

Para cualquier dominio (e incluso dynhost.es) podrás crear redirecciones de email, de modo que cualquier email recibido a través de esa cuenta, sea redirigido a donde tú quieras.

Utilizar Bases de Datos

Si tienes una web dinámica, necesitarás de una base de datos. Adquiriendo el paquete básico tienes disponible una base de datos MySQL.
