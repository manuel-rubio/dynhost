¿Qué es Dymmer?

Dymmer es un sitio que permite la creación de nombres de máquina, de modo que, si tienes un servidor o una máquina en Internet con IP dinámica (que cambia cada poco tiempo), puedes crear un nombre que se vaya actualizando con los cambios de IP que vaya teniendo tu máquina de cara a Internet.

Esto es útil para:

  • Tener localizada siempre la IP de tu casa.
  • Un nombre para acceder a tu servidor casero.
  • Un subdominio temporal para montar y presentar una web.
