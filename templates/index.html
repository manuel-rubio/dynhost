¿Qué es DynHost?

DynHost es un sitio que permite la creación de nombres de máquina, de modo que, si tienes un servidor o una máquina en Internet con IP dinámica (que cambia cada poco tiempo), puedes crear un nombre que se vaya actualizando con los cambios de IP que vaya teniendo tu máquina de cara a Internet.

Alta Gratuita

DynHost es un servicio gratuito, pero limitado, por cada usuario no se permiten crear más de 5 nombres de máquinas. Si quieres darte de alta solo tienes que seguir este enlace.

Uso profesional

DynHost ha agregado funcionalidades específicas para permitirte gestionar dominios reales, no solo dynhost.es. En general:

Gestionar Zonas DNS

Cualquier dominio que compres, podrás configurarlo desde DynHost, de modo que crear nombres como los típicos: www, mail, etc; será un trabajo trivial.

Crear Buzones de Correo

Podrás crear buzones de correo para poder acceder vía IMAP o POP3, enviar correo a través de SMTP e incluso conectarte a nuestro sistema de webmail para enviar y recibir emails.

Configurar Hosting web

Si necesitas espacio para albergar tu web, te ofrecemos un hosting para tu web, ya sea blog, o web corporativa, o foro, con esta herramienta podrás adecuarlo a tus necesidades.

Crear Redirecciones Web

Si tienes tu página creada en una herramienta de empresa externa y quieres que tu nuevo dominio la emplee, simplemente configurando esta redirección, el sitio web recibirá las peticiones desde el dominio que hayas adquirido.

Manejar Redirecciones de Correo

Para cualquier dominio (e incluso dynhost.es) podrás crear redirecciones de email, de modo que cualquier email recibido a través de esa cuenta, sea redirigido a donde tú quieras.

Utilizar Bases de Datos

Si tienes una web dinámica, necesitarás de una base de datos. En DynHost te damos la posibilidad de usar una base de datos 100% compatible con MySQL: MariaDB; con mejor rendimiento y más características que MySQL.

Acceder mediante FTP / SFTP

Si necesitas cargar de forma más fácil y segura la información dentro del hosting web, a través de cuentas FTP (o SFTP, agregando una capa de seguridad), podrás conseguirlo.
